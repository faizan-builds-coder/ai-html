# HTML Projects Documentation

## Project 1: freeCodeCamp Survey Form

### Description

This project is a simple survey form created using HTML, CSS, and JavaScript. The purpose of this form is to collect feedback from users about freeCodeCamp and understand how the platform can be improved.

### Features

* User can enter their name, email, and age.
* Dropdown menu to select the current role.
* Radio buttons to choose the likelihood of recommending freeCodeCamp.
* Dropdown menu to select the favorite feature of freeCodeCamp.
* Checkboxes to choose areas that need improvement.
* Submit button to collect form data.
* JavaScript displays all entered information in an alert box.

### Technologies Used

* HTML5
* CSS3
* JavaScript

### Code Explanation

#### HTML

The HTML structure creates the survey form and includes:

* Text input fields for name, email, and age.
* Select dropdowns for role and favorite feature.
* Radio buttons for recommendation options.
* Checkboxes for improvement suggestions.
* Submit button for form submission.

#### CSS

The CSS is used to:

* Create a clean and modern design.
* Center the form on the page.
* Add spacing, borders, and shadows.
* Style buttons, inputs, and dropdown menus.

#### JavaScript

JavaScript handles the form submission:

* Prevents the page from refreshing.
* Collects user input values.
* Retrieves selected radio button and checkboxes.
* Displays the collected information in an alert message.

### Learning Outcomes

* Creating forms in HTML.
* Styling forms with CSS.
* Handling form data using JavaScript.
* Working with radio buttons, checkboxes, and dropdown menus.

# Project 2: Timeless Watches Product Page

### Description

This project is a modern e-commerce product page for a luxury watch brand called "Timeless Watches". It showcases a watch product with images, pricing, ratings, quantity controls, and promotional offers.

### Features

* Responsive navigation bar.
* Product image gallery with thumbnails.
* Dynamic image switching.
* Product pricing and discount display.
* Star rating section.
* Quantity increase and decrease buttons.
* Add to Bag button.
* Promotional offer section.
* Clean and modern user interface.

### Technologies Used

* HTML5
* CSS3
* JavaScript

### Code Explanation

#### HTML

The HTML structure contains:

* Navigation bar with brand name and links.
* Search bar for product searching.
* Product image gallery.
* Product information section.
* Quantity controls.
* Add to Bag button.
* Promotional offer box.

#### CSS

The CSS is responsible for:

* Responsive page layout.
* Flexbox-based product design.
* Navigation bar styling.
* Product card appearance.
* Buttons, ratings, and offer box styling.
* Hover effects and visual enhancements.

#### JavaScript

##### changeQty() Function

This function:

* Increases or decreases the quantity value.
* Prevents the quantity from going below 1.

##### changeImg() Function

This function:

* Changes the main product image when a thumbnail is clicked.
* Highlights the selected thumbnail.

### Learning Outcomes

* Building a professional product page.
* Using Flexbox for layouts.
* Creating image galleries.
* Manipulating DOM elements with JavaScript.
* Implementing quantity controls and interactive features.
